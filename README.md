# 极简沉浸式节拍器 (Simple Immersive Metronome)

## 项目背景 (Motivation)
做这个项目的起因是因为市面上很难找到纯净且好用的节拍器应用（多数带有广告或功能过于繁杂）。因此，借助 AI 直接按照自己的需求生成了一个干净、专一的节拍器，旨在提供无干扰的沉浸式练琴体验。

## 项目信息 (Project Info)
本项目是一个基于 Flutter 构建的跨平台移动端应用。旨在提供最核心的节拍器功能，让你专注于音乐本身。

## 快速开始 (Getting Started)

### 拉取代码 (Clone)
```bash
git clone https://github.com/MrJoeYW/metronome_app.git
cd metronome_app
```

### 依赖安装 (Install Dependencies)
确保你的系统已经安装好了 [Flutter SDK](https://flutter.dev/docs/get-started/install)。
拉取代码后，在项目根目录运行以下命令获取依赖：
```bash
flutter pub get
```

### 启动运行 (Run)
连接上你的手机、开启模拟器或者选择桌面端为运行目标，然后执行：
```bash
flutter run
```

### 编译打包 (Build)
这里以 Android APK 为例（其他平台类似）：
```bash
flutter build apk
```
生成的 APK 文件可以在 `build/app/outputs/flutter-apk/app-release.apk` 找到。
