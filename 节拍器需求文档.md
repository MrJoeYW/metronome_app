# Flutter 节拍器项目开发需求文档 (Android)

## 1. 项目目标

开发一个具有**极低延迟**、**高精度计时**功能的安卓节拍器。界面风格要求现代化、简洁，且支持内置多音色库（包含人声数拍）。

## 2. 核心技术要求

- **开发环境**: Flutter (Dart)
- **音频引擎**: 必须使用支持低延迟的方案。推荐使用 `flutter_oboe` (基于 Google C++ Oboe 库) 或 Android 原生 `SoundPool` 接口。
- **计时精度**: 严禁仅使用 Dart 标准 `Timer`。需通过底层微秒级计时或音频回调循环来驱动，确保 BPM 在长时间运行下不产生累积误差。
- **目标平台**: 专门针对 Android 设备进行优化。

------

## 3. 功能模块详情

### 3.1 节拍引擎逻辑

- **BPM 计算公式**:

  $$T_{interval\_ms} = \frac{60,000}{BPM}$$

- **拍号支持**: 支持 1/4, 2/4, 3/4, 4/4, 6/8 等常用拍号。

- **稳定性保障**: 所有音频素材在应用启动时必须预加载至内存 Buffer。

### 3.2 内置音色库 (Built-in Assets)

系统仅使用预设的音源文件，不开放外部导入，以确保播放稳定性。音色分为以下组别：

- **基础音色**: 机械（Mechanical）、电子（Electronic）、木鱼/鼓棒（Wood）。
- **人声数拍 (Vocal Counting)**:
  - 支持**英文** (One, Two, Three, Four...) 和**中文** (一, 二, 三, 四...)。
  - 逻辑要求：每一拍触发时，根据当前节拍索引播放对应的数字音效。
- **强弱分配**: 允许用户自定义“强拍”与“弱拍”分别使用哪种内置音效。

### 3.3 UI 交互设计 (现代化设计)

- **主控界面**:
  - **大旋钮 (Knob)**: 屏幕中央用于调节 BPM 的无极旋钮，支持滑动交互。
  - **拍数选择**: 横向滚动条或卡片切换。
  - **可视化反馈**: 顶部设置与拍数相等的指示灯，随节拍高亮闪烁（强拍颜色需有区分）。
- **配色**: 采用深色模式 (Dark Mode)，配合荧光色（如电光绿或亮蓝）作为交互反馈色。

------

## 4. 安卓专项性能优化

- **低延迟路径**: 调用 Android 底层采样率（通常为 44.1kHz 或 48kHz），避免系统重采样。
- **后台运行**: 实现 `WakeLock` 和 `Foreground Service`，确保手机灭屏或切到后台时节拍不断流、不减速。
- **触感反馈**: 强拍触发时同步调用 Android 原生触感反馈（Haptic Feedback）。
- **音频焦点**: 妥善处理 `OnAudioFocusChangeListener`，在有电话或系统通知时正确挂起。

------

## 5. Codex 开发指令 (Prompt)

> **Role**: Senior Flutter & Android Developer.
>
> **Task**: Build a low-latency metronome app.
>
> **Requirements**:
>
> 1. **Audio**: Use `flutter_oboe` or native `SoundPool` to ensure sub-10ms audio latency on Android.
> 2. **Timing**: Implement a high-precision periodic trigger using microsecond timestamps to calculate $T = 60000 / BPM$.
> 3. **Vocal Counting**: Include built-in English and Chinese voice samples (1, 2, 3, 4...). Map samples to the current beat index based on the selected time signature (e.g., 4/4, 3/4).
> 4. **Sound Management**: Pre-load all wave files into memory buffers. No external file imports.
> 5. **UI**: Create a modern dark-themed UI with a large circular BPM slider and pulsating visual indicators for each beat.
> 6. **Performance**: Prevent the app from being throttled by the Android system during screen-off (use WakeLock/Foreground Service).

------

## 6. 资源命名规范

- 音频文件应存放在 `assets/sounds/` 目录下。
- **命名严禁使用点号**（除后缀外），统一使用下划线，例如：`vocal_en_one.wav`, `vocal_zh_one.wav`, `click_accent.wav`。
